# Ecommerce Frontend (React + Vite)

## Quick Start
1. Ensure the Flask backend is running at `http://localhost:5000/api`
2. Install deps
   ```bash
   npm install
   npm run dev
   ```
3. Visit http://localhost:3000

### Config
- API base is set in `src/config.js` (default `http://localhost:5000/api`)
- You can override via env: `VITE_API_BASE=http://127.0.0.1:5000/api npm run dev`
